
# Calculadora de pontos

Projeto para fins educacionais, que simula uma calculadora de pontos, com o objetivo de trabalhar uso do Javascript, assim como, usar function e configurar botões.
## Aprendizados

variáveis, funções e utilizar strings em Javascript.


## Funcionalidades

- botões:

coletar moedas
derrotar inimigos
resetar pontos
## Stack utilizada

**Front-end:** html, javascript, css


## Deploy

Para fazer o deploy desse projeto clone o epositório e execulte o arquivo index.html

```bash
git clone https://github.com/Cecilia5645/calculadora-de-pontos
```
